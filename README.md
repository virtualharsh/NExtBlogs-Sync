# NESTJS NOTES

To make a new NEST PROJECT

```bash
npm i -g @nestjs/cli
```

```bash
nest new project-name
```

To Add a new controller

```bash
nest g controller controller-name
```

To Add a new service

```bash
nest g service service-name
```

To Create a new module

```bash
nest g module module-name
```

To create controller/service under module

```bash
nest g controller/service module/controller
```

To create everything

```bash
nest g resource module_name
```

To create everything nested

```bash
nest g resource module_name/module_name
```
